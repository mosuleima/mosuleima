## Featured Projects

### 180nm CMOS Charge-Pump Voltage Doubler

Designed a charge-pump voltage doubler for low-voltage power management using **Cadence Virtuoso**.
The project included schematic design, circuit simulations, layout implementation, DRC/LVS, and post-layout verification.

### Hardware Formal Verification

Applied formal verification methodologies using **SystemVerilog Assertions (SVA)** and **Cadence JasperGold** to verify hardware behavior beyond simulation coverage.

### High Step-Up DC-DC Boost Converter

Designed and optimized a DC-DC boost converter achieving **5V-to-50V conversion** with over **90% simulated efficiency**, including LTspice simulations, OrCAD schematics, gate-driver design, and snubber circuits.

### Digital Systems / FPGA Design

Designed and implemented digital systems using **Verilog/SystemVerilog**, finite state machines, simulation, debugging, and FPGA-based integration.
